P.U.L.S.E. — Deploy & Present Guide
====================================

DEPLOY TO VERCEL (2 minutes):
1. Go to vercel.com → Add New → Project → or just drag this whole
   folder onto the Vercel dashboard (vercel.com/new)
2. Name it something clean: pulse-forces
3. Deploy. Copy the live URL and send it back to Claude
   so the QR code can be embedded.

PRESENT (offline-safe):
- Just double-click index.html — it runs fully offline.
  (Fonts fall back to system fonts without internet; everything else works.)

PRESENTER CONTROLS:
- Arrow keys / Page Down / Space  →  jump to next section
- Arrow up / Page Up              →  previous section
- Home / End                      →  first / last section
- Press P                         →  hide/show the gold [PHYSICAL MODEL] cue tags
                                      (hide them if you don't want the class to see your cues)
